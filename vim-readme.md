# Shortcuts
## Editor
- :w = save
- :q = quit
- :q! = quit without save
- mk = Create mark called k
- d'k = Delete/cut until mark called k
- y'k = Copy until mark called k
- yyp = Duplicate line
- yy = Copy
- p = Paste

## NerdTree
- CTRL + W + W = Toggle between NERDTree & buffer
- ? (on NERDTree) = See NERDTree Help

## CTRLP.vim
- CTRL + P = Toogle fast search
